# Flash Flow Web Application

This is the high-performance React 19 frontend for the Flash Flow system, built with TanStack Start and Vite.

## Key Features

- **Premium UX**: Smooth animations with Lucide icons and Tailwind CSS v4 styling.
- **Real-time Updates**: Status polling for live stock and sale state rendering.
- **Unified Error Handling**: Premium modal overlay system for critical failures.
- **Layout Optimization**: Aggressively optimized for a single-screen 1080p desktop experience.
